# Carlos Lozano | Fullstack Portfolio

Portfolio personal desarrollado con un enfoque en la **robustez técnica** y la **resolución de problemas**. Esta web sirve como escaparate de mis proyectos y mi stack tecnológico actual.

## Tech Stack

El proyecto utiliza tecnologías modernas para garantizar un rendimiento óptimo y una experiencia de usuario fluida:

* **Core:** React + Vite (o Next.js)
* **Estilos:** Tailwind CSS (Mobile-first design)
* **Iconos:** React Icons
* **Animaciones:** Framer Motion
* **Estado & Tema:** Hooks nativos (Dark/Light mode)

## Características principales

* **Modo Oscuro/Claro:** Implementado con Tailwind CSS y persistencia de preferencia.
* **Diseño Responsivo:** Adaptado para una navegación perfecta en cualquier dispositivo.
* **UI Interactiva:** Detalles dinámicos como efectos de scroll, transiciones suaves y un sistema de "nieve" interactivo en el Hero.
* **Gestión de Proyectos:** Estructura modular para visualizar proyectos destacados, otros proyectos y trabajos en desarrollo (WIP).
